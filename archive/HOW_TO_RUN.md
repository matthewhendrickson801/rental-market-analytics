# How to Run the Analysis Scripts

## Important Note About File Paths

The scripts in the `scripts/` directory were originally written to run from the root directory. Since we've organized them into subdirectories, you have two options:

## Option 1: Run from Root Directory (Recommended)

All scripts are designed to be run from the project root directory:

```bash
# From project root directory

# Phase 1: Data Extraction & Cleaning
python3 scripts/data_extraction/combine_data.py
python3 scripts/data_extraction/missing_values_analysis.py
python3 scripts/data_extraction/analyze_missing_rent.py
python3 scripts/data_extraction/remove_missing_rent_zips.py

# Phase 2: Statistical Analysis
python3 scripts/analysis/generate_eda_statistics.py
python3 scripts/analysis/univariate_analysis.py
python3 scripts/analysis/bivariate_analysis.py
python3 scripts/analysis/comprehensive_heatmap.py
python3 scripts/analysis/deep_insights_analysis.py

# Phase 3: Feature Engineering
python3 scripts/feature_engineering/create_mismatch_indexes.py
python3 scripts/feature_engineering/create_rent_waste_index.py
python3 scripts/feature_engineering/create_city_boom_index.py
```

## Option 2: Use Pre-Generated Results

All analysis has already been completed! You can find:

- **Final Dataset**: `data/final/final_dataset_with_boom_index.csv`
- **Visualizations**: `visualizations/*.png`
- **Reports**: `reports/Matthew_EDA_Complete_Report.md`

## File Locations

### Input Files (Required for scripts to run)
- Raw data: `data/raw/*.zip`
- Processed data: `data/processed/*.csv`

### Output Files (Generated by scripts)
- Final datasets: `data/final/*.csv`
- Visualizations: `visualizations/*.png`
- Analysis summaries: `data/processed/*.csv`

## Troubleshooting

If you get "File not found" errors:
1. Make sure you're running from the project root directory
2. Check that the data files exist in `data/raw/`
3. Verify Python packages are installed

The `archive/` directory contains old/duplicate files from the development process. These are not needed for the final submission but are kept for reference.
